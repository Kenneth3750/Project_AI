          Tips for Using NAIA

          To ensure the best experience with NAIA, please follow these recommendations:

          Use Chrome Browser

          For optimal performance, use Google Chrome as your primary browser to access NAIA. Some features, like Web Speech API, may not be available in other browsers, which could affect the app's functionality.

          Set Browser Language

          Set your browser's default language to match the language you intend to use with NAIA. For example, if you want to interact with the avatar in English, set your browser's language to English for the best results. This applies to other supported languages as well.

          Manage Lighting

          Avoid areas with strong backlighting when interacting with NAIA. Excessive light behind you can interfere with the facial recognition feature during your interaction with the avatar. Choose a well-lit environment where the light source is in front of you for best results.

          Minimize Background Noise

          Use NAIA in a quiet environment. Excessive background noise or multiple voices can interfere with the speech-to-text module, affecting the accuracy of voice interactions. Choose a quiet space for the best voice recognition performance.
